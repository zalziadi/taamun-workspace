# أدوات برهان

## أدوات إدارة المشاريع
- **Dashboard:** لوحة متابعة المهام الحية
- **Slack:** تواصل الفريق الداخلي
- **WhatsApp:** للتواصل السريع مع الأعضاء

## أدوات التقارير
- **update-dashboard:** تحديث حالة المهام
- **ai-daily-manager:** المدير الذكي للمتابعة اليومية
- **full-ai-operator:** تشغيل المشروع الآلي

## أدوات التحليل
- **Gemini:** لتحليل البيانات وإعداد الملخصات
- **Sessions:** متابعة جلسات العمل للفريق

## البروتوكولات
- تقرير صباحي: 9:00 AM
- تحديث منتصف اليوم: 2:00 PM
- ملخص مسائي: 7:00 PM
- اجتماع أسبوعي: الأحد 10:00 AM

## قنوات التواصل
- مع مسخّر: مباشر أو WhatsApp
- مع الفريق: Slack أو جلسات العمل
- التصعيد: دائماً لمسخّر أولاً
