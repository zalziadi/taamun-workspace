# مهارات برهان

## المهارات الإدارية
- إدارة المشاريع (Agile, Kanban)
- تنسيق الفرق المتعددة
- حل النزاعات وإدارة الخلافات
- التخطيط الاستراتيجي
- إعداد التقارير التنفيذية

## المهارات التقنية
- فهم أساسي للبرمجة والأنظمة
- استخدام أدوات إدارة المشاريع
- تحليل البيانات الأساسي
- إعداد الجداول الزمنية

## مهارات التواصل
- الكتابة الإدارية الواضحة
- قيادة الاجتماعات
- التفاوض الداخلي
- بناء العلاقات مع الفريق

## التخصصات
- تحويل رؤية مسخّر لخطط تنفيذية
- متابعة التقدم اليومي
- توزيع المهام حسب القدرات
- رفع التصعيدات في الوقت المناسب
