# بيئة العمل المتزامنة - عيدية تمعّن

## الأدوات المتكاملة

### 1. OpenClaw (الأساس)
- المحرك الرئيسي للذكاء الاصطناعي
- إدارة الإيجنتات والمهام
- التواصل مع القنوات المختلفة

### 2. Claude Code (التطوير)
- كتابة وتطوير الأكواد
- مراجعة البرمجيات
- حل المشاكل التقنية المعقدة

### 3. Cursor (المحرر الذكي)
- تحرير الملفات بذكاء اصطناعي
- التكامل مع Git
- البرمجة التعاونية

### 4. Obsidian (الذاكرة المرئية)
- توثيق المعرفة
- ربط الأفكار والمفاهيم
- البحث السريع
- المزامنة عبر الأجهزة

## بروتوكول العمل المتزامن

### المرحلة 1: التخطيط (OpenClaw)
```bash
# مسخّر يستقبل المتطلبات
# برهان ينشئ خطة العمل
# توزيع المهام على الفريق
```

### المرحلة 2: التنفيذ (Multi-Agent)
```bash
# شام + CJ → Claude Code للتطوير
# وردة → Cursor لتحسين التجربة
# الجميع → Obsidian للتوثيق
```

### المرحلة 3: المراجعة (Sync)
```bash
# مزامنة التحديثات
# مراجعة الأكواد
# تحديث الوثائق
```

## تكامل الأدوات

### OpenClaw ↔ Claude Code
```bash
# لتشغيل Claude Code من OpenClaw
sessions_spawn runtime:"acp" agentId:"claude-code" task:"..."
```

### OpenClaw ↔ Cursor
```bash
# فتح مشروع في Cursor
exec command:"cursor /path/to/project"
```

### OpenClaw ↔ Obsidian
```bash
# إنشاء ملاحظة جديدة
write path:"/path/to/obsidian/vault/note.md" content:"..."

# فتح Obsidian
exec command:"open -a Obsidian"
```
