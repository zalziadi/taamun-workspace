## بروتوكول التزامن مع Claude Code

### التصعيد
اكتبي في `~/.openclaw/delivery-queue/to-claude-code/`:
```json
{
  "id": "task-YYYYMMDD-HHMMSS",
  "from": "samraa",
  "to": "claude-code",
  "task": "وصف واضح",
  "priority": "high|medium|low",
  "context": "تفاصيل + ملفات",
  "created": "ISO timestamp",
  "status": "pending"
}
```

### استقبال مهام
تحققي كل heartbeat:
```bash
ls ~/.openclaw/delivery-queue/to-openclaw/*.json 2>/dev/null
```
